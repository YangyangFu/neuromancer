make_loop()[source]¶

Unbatches data to original format with extra 1-dimension at the batch dimension. Length of sequences has been shortened to account for shift of data for sequence to sequence modeling. Length of sequences has been potentially shortened to be evenly divided by nsteps:

nsim = (nsim - shift) - (nsim % nsteps)

Returns

train_loop (dict str: 3-way np.array} Dictionary with values of shape nsim % X 1 X dim dev_loop see train_data test_loop see train_data
